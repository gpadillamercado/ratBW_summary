
# Digitized Rat Body Weight Measurements over Time

<!-- badges: start -->
<!-- badges: end -->

In this repo I have digitized a summary of body weight measurements over the lifetime
of laboratory rats as seen in the document 
_RECOMMENDATIONS FOR AND DOCUMENTATION OF BIOLOGICAL VALUES FOR USE IN RISK ASSESSMENT_
published by the US EPA in 1988. A copy of the original text file or OCR data is found in 
`500022JL.txt`. The script I used for programmatic and manual extraction & digitization
of the data is found in `extraction.R`. The final result of this digitization is found
in a TSV file: `ratBW_data1.tsv`
